Algorithm for word count:
1. Get the file name from user
2. Go to the location where the input files are stored
3. start counting the words
   3.a consider it as a word where there is a space between two characters
   3.b consider it as a word where there is a new line character at the end
4. Store the count in output.txt file
5. Start counting the special characters
  5.a Pick a character and check if its a special character
6. Start counting the line count
   6. a consider it as a line if it ends \n
   6. b Ignore if there is no alphabets because its a empty line
7. Start counting the paragraph
   7. a consider it as a paragraph when there is one of more non-empty lines separated by one or more empty lines.
